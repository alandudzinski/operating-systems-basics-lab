# Windows vs Linux CLI Cheat Sheet

## Navigation
| Task | Windows | Linux |
|---|---|---|
| List files | `dir` | `ls` |
| Change directory | `cd folder` | `cd folder` |
| Go back one directory | `cd ..` | `cd ..` |
| Show current directory | `cd` | `pwd` |
| Clear screen | `cls` | `clear` |

## System Information
| Task | Windows | Linux |
|---|---|---|
| Current user | `whoami` | `whoami` |
| Hostname | `hostname` | `hostname` |
| Network info | `ipconfig` | `ip addr` |
| System info | `systeminfo` | `uname -a` |
| Running processes | `tasklist` | `ps aux` |

## File Commands
| Task | Windows | Linux |
|---|---|---|
| Create folder | `mkdir folder` | `mkdir folder` |
| Delete file | `del file.txt` | `rm file.txt` |
| Read file | `type file.txt` | `cat file.txt` |
| Copy file | `copy file.txt backup.txt` | `cp file.txt backup.txt` |
| Move file | `move file.txt folder` | `mv file.txt folder` |
